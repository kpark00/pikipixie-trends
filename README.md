# 🎀 피키픽시 트렌드 워치

피키픽시 헤어 코스메틱 브랜드 마케팅 참고용 — 헤어 관련 YouTube Shorts / TikTok / Instagram Reels를
매일 수집해 조회수 구간별로 조회하고, 매주 마케팅 소구점 리포트를 자동 생성하는 사이트입니다.
**API 키 불필요, 별도 설치 불필요** (표준 python3만 사용).

## 두 가지 사용 방법

### ① 웹 버전 (GitHub Pages) — 아무 것도 실행할 필요 없음
- GitHub Actions가 **매일 05:30(한국시간)** 자동으로 데이터를 수집해 사이트에 반영
- **매주 월요일 07:00** 주간 마케팅 소구점 리포트 자동 생성 (📊 주간 리포트 버튼)
- 웹 버전은 읽기 전용: 인스타 수동 등록/조회수 보정은 브라우저(localStorage)에만 저장됨
- GitHub 저장소의 Actions 탭에서 "Run workflow"로 즉시 수집을 트리거할 수도 있음

### ② 로컬 버전 — 즉시 수집·리포트 생성 가능
```bash
python3 server.py
```
브라우저가 자동으로 http://localhost:8787 을 엽니다.
- **"↻ 오늘 데이터 수집"**: 즉시 최신 데이터 수집 (1~2분)
- **"📊 주간 리포트"**: 누를 때마다 현재 데이터로 리포트를 새로 생성
- **"⚙️ 설정"**: 키워드, 수집 기간, 국내 필터 수정

## 플랫폼별 수집 방식

| 플랫폼 | 방식 | 조회수 |
|---|---|---|
| YouTube Shorts | 유튜브 웹 내부 검색(Innertube), 키 불필요. 한국 기준, 조회수순, 3분 이하 | 자동 |
| TikTok | tikwm.com 공개 API. 한국(KR) 지역 + 한글 제목/채널만 | 자동 |
| Instagram Reels | 공개 검색엔진(Brave→DuckDuckGo→Mojeek)에서 릴스 URL·제목 자동 발굴 | 수동 보정 |

- **국내 위주**: 제목·채널에 한글이 없는 해외 영상은 제외 (설정에서 해제 가능)
- **Instagram 제약**: 로그인 없이 조회수를 공개하지 않아 조회수 자동 수집은 불가.
  카드의 **"조회수 입력"** 버튼으로 앱에서 본 수치를 직접 기록.
  검색엔진이 봇을 차단하면 발굴이 일시적으로 실패할 수 있음 (다음 수집에서 재시도).

## 주간 마케팅 소구점 리포트 (reports/)

수집 데이터를 분석해 매주 자동 생성:
1. 플랫폼 요약 (수집량·최고·중앙값 조회수)
2. 이번 주 조회수 TOP 10
3. 급상승·신규 주목 영상 (일별 조회수 이력 기반)
4. 테마별 소비자 관심 (손상모 복구·윤기·볼륨·두피·염색·가성비 등 9개 테마, 조회수 가중 점유율)
5. 훅(Hook) 패턴 분석 (리스트형·질문형·경고형·비포애프터형 등)
6. **피키픽시 소구점 제안** — 상위 테마별 방향·근거·카피 시안
7. 해시태그 TOP 20

모든 수치는 수집된 데이터에서 계산하며 외부 추정치를 포함하지 않습니다.

## 파일 구조

```
index.html      # 프론트엔드 (로컬/웹 겸용)
server.py       # 로컬 웹서버 (표준 라이브러리만)
fetch.py        # 수집 엔진 (YouTube/TikTok/Instagram)
report.py       # 주간 리포트 생성기
config.json     # 키워드·필터 설정
data/videos.json    # 누적 수집 데이터 (조회수 일별 이력 포함)
reports/            # 주간 리포트 (Markdown)
.github/workflows/  # 매일 수집·매주 리포트 자동화
```

## 주의

- TikTok(tikwm)·Instagram 발굴은 비공식 경로라 언제든 막힐 수 있습니다. 막히면 해당 플랫폼만
  실패하고 나머지는 정상 동작합니다.
- 이 저장소는 공개(public)입니다 — GitHub Pages 무료 호스팅 조건. 수집 데이터는 모두 공개
  플랫폼의 공개 정보입니다.
